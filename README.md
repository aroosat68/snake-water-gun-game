Snake–Water–Gun (Python Game)

A simple Snake–Water–Gun game written in Python using the random module.

How to Play:

Run the file snake_water_gun.py

Enter your choice as:

s → Snake

w → Water

g → Gun

Computer will choose randomly.

The result will be displayed.

Game Rules

Snake drinks Water → Snake wins

Water puts out Gun → Water wins

Gun shoots Snake → Gun wins

Same choice → Draw

Run the Game Locally:

Make sure Python is installed on your system, then run:

python snake_water_gun.py
